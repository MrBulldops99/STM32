# STM32
STM32 microcontroller implemenation.
STM32 implementation roadmap
  Set up the STM32
  Blink 
  Serial Communication Protocols: UART,SPI,CAN,I2C
  Wireless Communication Protocols: Wi-Fi,Bluetooth etc.
  Incorporate sensor
    Passive reading (example: Temp sensor, Pressure sensor)
    Active reading (example: Button presses)
  Peripherals
    GPIO
    Timers
    Interrupts
  Bare-metal implementation
  RTOS
    Task scheduling
    Semaphores
